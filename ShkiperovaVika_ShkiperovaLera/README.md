# TODO API (Flask + Docker)

HTTP API для управления списком задач (TODO) с приоритетом, сроком и статусом. 
Приложение реализовано на *Flask* и упаковано в Docker, запуск осуществляется через `docker compose`.

## *Функциональность*

API позволяет:

* получать список задач с фильтрацией по статусу и приоритету;
* создавать новую задачу с названием, сроком и приоритетом;
* изменять статус существующей задачи;
* удалять задачу по идентификатору.

Данные хранятся в памяти процесса (список `tasks`), поэтому при перезапуске контейнера они сбрасываются — этого достаточно для учебного задания.
